# Simulation-Based Power Analysis for DAISIE

This repository contains R code for simulating island biodiversity datasets and assessing the performance of the DAISIE model under incomplete and different sampling methods.
The simulations explore how data coverage and sampling strategies affect the reliability of model parameter inference.

## Features
- Simulate complete island datasets of various sizes
- Subsample species at different fractions to mimic incomplete sampling
- Introduce targeted biases (lineage diversity, species age, mainland species)
- Evaluate inference reliability and data requirements
